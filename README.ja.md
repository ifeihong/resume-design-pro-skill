### 2. AI コンテンツ最適化エンジン — 真実性優先

あなたのコンテンツは**プロフェッショナルに書き換えられます** — あなたの実際の情報に基づき、決して捏造しません：

| 最適化前 | 最適化後（データ提供あり） | 最適化後（データなし） |
|---------|------------------------|----------------|
| "ウェブサイト開発を担当" | "会社のウェブサイトのフルスタック開発を主導し、コンバージョン率を 35% 向上" | "会社のウェブサイトのフルスタック開発を主導し、大きなビジネスインパクトを実現" |
| "チームワークが得意" | "8 人以上のクロスファンクショナルチームと協力し、3 つの主要リリースを成功させた" | "クロスファンクショナルチームと協力し、複数の主要リリースを成功させた" |
| "React を素早く習得" | "2 週間で React エコシステムを習得し、第 3 スプリントで本番機能をリリース" | "React エコシステムを迅速に習得し、スケジュールを前倒しで本番機能をリリース" |

**9 ステップ最適化プロセス：**
1. **抽出** — 生の情報を解析し、各データの出典を記録
2. **保持** — すべての職務経歴を保持（決して削除しない）
3. **階層化** — 関連性で優先順位付け：コア（3-5 ハイライト）/ サポート（2-3）/ 早期（1-2）
4. **定量化（誠実）** — 3段階戦略：正確なデータ保持 / 曖昧なデータは推定として明記 / データなしは定性的記述に変更
5. **帰属ロック** — 各説明が正しい企業に属していることを確認
6. **言い換え** — アクション動詞と STAR メソッドを使用
7. **再構築** — プロフェッショナルな履歴書形式
8. **HR チェック** — 実際の採用マネージャーの審査基準に照らして自己検証
9. **確認** — 透明性のあるビフォー/アフターを表示し、あなたが承認

**核心原則：**
- **決してデータを捏造しない** — 提供されていない数字は想像しない
- **すべての経歴を保持** — すべての職務経歴が履歴書に残る
- **企業帰属ロック** — 説明が企業間で混同されることはない
- **コンテンツ密度の選択** — 簡潔版（1 ページ）/ 標準版（1-2 ページ）/ 詳細版（2-3 ページ）

**元のコンテンツを保持したい？** `/raw` と入力するだけ — ゼロ変更、純粋なデザイン変換。

### 3. 完全な出力パッケージ

1 回の呼び出しで、**6 つの成果物**：

| ファイル | 用途 |
|---------|------|
| `resume-{名前}-{スタイル}.html` | 美しいビジュアル履歴書 — ブラウザで開く |
| `resume-{名前}-{スタイル}.pdf` | **自動エクスポート PDF** — ピクセル完璧、印刷可能 |
| `resume-{名前}-ats.html` | プレーンテキスト ATS バージョン — 採用管理システムに安全 |
| `resume-{名前}.md` | Markdown バックアップ — いつでも編集・再利用可能 |
| `cover-letter-{名前}.html` | マッチングカバーレター — 同じビジュアルスタイル |
| `linkedin-export.md` | LinkedIn 最適化コンテンツ — コピー＆ペーストですぐ使用 |

**PDF は自動生成されます**：Playwright ヘッドレスブラウザを使用して自動エクスポート。色、フォント、グラデーション、レイアウトはブラウザ表示と完全に一致します。Ctrl+P を押す必要はありません。

## クイックスタート

### ユーザー向け

```bash
# 任意の AI コーディングアシスタントで
use resume-design-pro
```

プロンプトに従って：
1. 履歴書情報を提供（アップロード、貼り付け、または対話）
2. コンテンツ最適化モード（デフォルト）または RAW モードを選択
3. 11 種類のデザインスタイルから選択
4. 完全な履歴書パッケージを取得

### 開発者向け

```bash
# クローンしてインストール
git clone <repo-url>
cd resume-design-pro/scripts
npm install

# HTML から PDF をエクスポート
node export-pdf.js ../templates/template-01-minimalism.html
```

## PDF エクスポート

### 方法 1：ブラウザ印刷（最も簡単）
1. ブラウザで HTML ファイルを開く
2. Ctrl+P → 「PDF として保存」を選択
3. 用紙サイズ：A4、余白：デフォルト
4. 「背景のグラフィック」をチェックして色を保持

### 方法 2：Node.js スクリプト
```bash
cd scripts
npm install
node export-pdf.js ../templates/template-01-minimalism.html
```

## プラットフォーム互換性

**Resume Design Pro** はオープンな SKILL 形式で構築されています — Skill 呼び出しをサポートする任意の AI アシスタントで動作します。

| カテゴリ | 例 |
|----------|-----|
| AI コーディングアシスタント | Claude Code、Trae、Codex、Cursor、GitHub Copilot Chat |
| AI Agent プラットフォーム | Coze (扣子)、Dify、LangChain Agents、AutoGen |
| カスタム AI ツール | Markdown ベースの Skill ファイルをサポートする任意のプラットフォーム |

**要件：** プラットフォームは以下のみが必要です：
1. Markdown 指示の読み取りと実行
2. テキスト/HTML 出力の生成
3. 基本的なユーザーインタラクションのサポート（プロンプト → レスポンス）

特殊なプラグイン、API、統合は不要です。
