### 2. AI コンテンツ最適化エンジン

あなたのコンテンツは**プロフェッショナルに書き換えられます** — 単なるフォーマットではなく：

| 最適化前 | 最適化後 |
|---------|---------|
| "ウェブサイト開発を担当" | "会社のウェブサイトのフルスタック開発を主導し、コンバージョン率を 35% 向上" |
| "チームワークが得意" | "8 人以上のクロスファンクショナルチームと協力し、3 つの主要リリースを成功させた" |
| "React を素早く習得" | "2 週間で React エコシステムを習得し、第 3 スプリントで本番機能をリリース" |

**6 ステップ最適化プロセス：**
1. **抽出** — 生の情報を解析
2. **フィルタリング** — 高インパクトなコンテンツのみ保持
3. **定量化** — 指標と数字を追加
4. **言い換え** — アクション動詞と STAR メソッドを使用
5. **再構築** — プロフェッショナルな履歴書形式
6. **確認** — ビフォー/アフターを表示し、あなたが承認

**元のコンテンツを保持したい？** `/raw` と入力するだけ — ゼロ変更、純粋なデザイン変換。

### 3. 完全な出力パッケージ

1 回の呼び出しで、**5 つの成果物**：

| ファイル | 用途 |
|---------|------|
| `resume-{名前}-{スタイル}.html` | 美しいビジュアル履歴書 — ブラウザで開き、PDF として印刷 |
| `resume-{名前}-ats.html` | プレーンテキスト ATS バージョン — 採用管理システムに安全 |
| `resume-{名前}.md` | Markdown バックアップ — いつでも編集・再利用可能 |
| `cover-letter-{名前}.html` | マッチングカバーレター — 同じビジュアルスタイル |
| `linkedin-export.md` | LinkedIn 最適化コンテンツ — コピー＆ペーストですぐ使用 |

## クイックスタート

### ユーザー向け

```bash
# Claude Code / Trae / Codex で
use resume-design-pro
```

プロンプトに従って：
1. 履歴書情報を提供（アップロード、貼り付け、または対話）
2. コンテンツ最適化モード（デフォルト）または RAW モードを選択
3. 11 種類のデザインスタイルから選択
4. 完全な履歴書パッケージを取得

### 開発者向け

```bash
# クローンしてインストール
git clone <repo-url>
cd resume-design-pro/scripts
npm install

# HTML から PDF をエクスポート
node export-pdf.js ../templates/template-01-minimalism.html
```

## PDF エクスポート

### 方法 1：ブラウザ印刷（最も簡単）
1. ブラウザで HTML ファイルを開く
2. Ctrl+P → 「PDF として保存」を選択
3. 用紙サイズ：A4、余白：デフォルト
4. 「背景のグラフィック」をチェックして色を保持

### 方法 2：Node.js スクリプト
```bash
cd scripts
npm install
node export-pdf.js ../templates/template-01-minimalism.html
```
