### 2. AI 内容深度优化引擎 — 真实性优先

你的内容会被**专业级重写** — 基于你的真实信息，绝不编造数据：

| 优化前 | 优化后（有你提供的数据） | 优化后（无数据） |
|--------|------------------------|----------------|
| "负责网站开发" | "主导公司网站全栈开发，转化率提升 35%" | "主导公司网站全栈开发，带来显著业务增长" |
| "擅长团队合作" | "与 8 人以上跨职能团队协作，成功交付 3 个重大版本" | "与跨职能团队紧密协作，成功交付多个重大版本" |
| "快速学会了 React" | "2 周内掌握 React 生态，第 3 个冲刺即交付生产级功能" | "快速掌握 React 生态，提前交付生产级功能" |

**9 步优化流程：**
1. **提取** — 解析你的原始信息，标注每条数据来源
2. **保留** — 保留全部工作经历，绝不删除任何一段
3. **分级** — 按相关性分层：核心经历（3-5 个亮点）/ 支撑经历（2-3 个）/ 早期经历（1-2 个）
4. **量化（真实）** — 三档策略：精确数据保留 / 模糊数据标注估算 / 无数据改用定性描述
5. **归属锁定** — 确保每条描述始终归属正确的公司，绝不张冠李戴
6. **润色** — 使用专业动词和 STAR 法则
7. **重构** — 专业简历格式
8. **HR 校验** — 对照真实招聘经理评审标准自检
9. **确认** — 透明展示前后对比，由你确认

**核心原则：**
- **绝不编造数据** — 你没提供的数字，我们绝不臆造
- **保留全部经历** — 你有的每段工作经历都会出现在简历上
- **公司归属锁定** — 描述绝不会在不同公司间混淆
- **内容密度可选** — 精简版（1 页）/ 标准版（1-2 页）/ 详细版（2-3 页）

**想保留原始内容？** 输入 `/raw` — 零修改，纯设计转换。

### 3. 完整输出包

一次调用，**6 份交付物**：

| 文件 | 用途 |
|------|------|
| `resume-{姓名}-{风格}.html` | 精美视觉简历 — 浏览器打开预览 |
| `resume-{姓名}-{风格}.pdf` | **自动导出 PDF** — 像素级还原，直接打印投递 |
| `resume-{姓名}-ats.html` | 纯文本 ATS 版本 — 招聘系统安全投递 |
| `resume-{姓名}.md` | Markdown 备份 — 随时编辑复用 |
| `cover-letter-{姓名}.html` | 配套求职信 — 同风格视觉 |
| `linkedin-export.md` | LinkedIn 优化内容 — 复制粘贴即用 |

**PDF 自动生成**：使用 Playwright 无头浏览器自动导出，颜色、字体、渐变、布局与网页版完全一致。无需打开浏览器按 Ctrl+P。

## 快速开始

### 用户使用

```bash
# 在任何 AI 编程助手中
use resume-design-pro
```

按提示操作：
1. 提供简历信息（上传、粘贴或对话）
2. 选择内容优化模式（默认）或原封不动模式
3. 从 29 种设计风格中选择
4. 获取完整简历包

### 开发者使用

```bash
# 克隆并安装
git clone <repo-url>
cd resume-design-pro/scripts
npm install

# 从 HTML 导出 PDF
node export-pdf.js ../templates/template-01-minimalism.html
```

## PDF 导出

### 方法 1：浏览器打印（最简单）
1. 用浏览器打开 HTML 文件
2. 按 Ctrl+P → 选择"另存为 PDF"
3. 纸张大小：A4，边距：默认
4. 勾选"背景图形"以保留颜色

### 方法 2：Node.js 脚本
```bash
cd scripts
npm install
node export-pdf.js ../templates/template-01-minimalism.html
```

## 平台兼容性

**Resume Design Pro** 基于开放的 SKILL 格式构建 — 可在任何支持 Skill 调用的 AI 助手上运行。

| 类别 | 示例 |
|------|------|
| AI 编程助手 | Claude Code、Trae、Codex、Cursor、GitHub Copilot Chat |
| AI Agent 平台 | 扣子 (Coze)、Dify、LangChain Agents、AutoGen |
| 自定义 AI 工具 | 任何支持 Markdown 格式 Skill 文件的平台 |

**平台要求：** 只需满足以下条件：
1. 读取并遵循 Markdown 指令
2. 生成文本/HTML 输出
3. 支持基础用户交互（提示 → 回复）

无需特殊插件、API 或集成。

## 适合谁用？

- **求职者**：技术、设计、金融、咨询领域
- **国际求职者**：申请跨国公司职位
- **转行者**：需要快速制作专业简历
- **学生**：申请实习或第一份工作
- **自由职业者**：希望以独特视觉形象脱颖而出
