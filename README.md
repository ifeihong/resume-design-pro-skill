### 2. AI Content Optimization Engine -- Truth-First

Your content gets **professionally rewritten** -- based on your real information, never fabricated:

| Before | After (with your data) | After (no data provided) |
|--------|------------------------|--------------------------|
| "Responsible for website development" | "Led full-stack development of company website, increasing conversion by 35%" | "Led full-stack development of company website, delivering significant business impact" |
| "Good at teamwork" | "Collaborated with cross-functional teams of 8+ members to deliver 3 major releases" | "Collaborated with cross-functional teams to deliver multiple major releases" |
| "Learned React quickly" | "Mastered React ecosystem in 2 weeks, shipped production feature in sprint 3" | "Rapidly mastered React ecosystem and shipped production features ahead of schedule" |

**The 10-step optimization process:**
1. **Extract** -- Parse your raw information with source attribution
2. **Preserve** -- Keep ALL your work experiences (never discard any)
3. **Tier** -- Prioritize by relevance: core (3-5 highlights) / support (2-3) / early (1-2)
4. **Quantify (Truthful)** -- Three-tier strategy: exact data / estimated (labeled) / qualitative (no fabrication)
5. **Lock Attribution** -- Ensure every description stays with its correct company
6. **Reword** -- Use action verbs and STAR methodology
7. **Restructure** -- Professional resume format
8. **HR Check** -- Validate against real hiring manager standards
9. **Confirm** -- Show before/after with transparency, you approve
10. **JD Match** -- Extract keywords from target job description, analyze coverage gaps, generate optimization suggestions for better alignment

**Key principles:**
- **Never fabricate data** -- if you didn't provide a number, we won't invent one
- **Preserve all experiences** -- every job you've had stays on your resume
- **Company attribution lock** -- descriptions never get mixed between companies
- **Content density choice** -- Concise (1 page) / Standard (1-2 pages) / Detailed (2-3 pages)

**Prefer to keep your original content?** Just say `/raw` -- zero modifications, pure design transformation.

### 3. Complete Output Package

One invocation, **10 deliverables**:

| File | Purpose |
|------|---------|
| `resume-{name}-{style}.html` | Stunning visual resume -- open in browser |
| `resume-{name}-{style}.pdf` | **Auto-exported PDF** -- pixel-perfect, print-ready |
| `resume-{name}-{style}.png` | **High-resolution PNG** -- 2x A4 screenshot, perfect for sharing on social media or messaging apps |
| `resume-{name}-ats.html` | Plain-text ATS version -- safe for applicant tracking systems |
| `resume-{name}.md` | Markdown backup -- easy to edit and reuse |
| `cover-letter-{name}.html` | Matching cover letter -- same visual style |
| `linkedin-export.html` | LinkedIn-optimized content -- copy-paste ready |
| `job-search-kit.html` | **Job search strategy report** -- target companies, timeline, salary reference, communication templates |
| `interview-prep.html` | **Interview preparation report** -- anticipated questions with answer frameworks, company intel |
| `resume-diagnosis.html` | **Resume diagnosis report** -- JD matching analysis with 6-dimension scoring and AI optimization suggestions |

**PDF is generated automatically** using Playwright headless Chromium -- colors, fonts, gradients, and layout are preserved exactly as seen in the browser. No need to open Chrome and press Ctrl+P.

**PNG export** -- High-resolution PNG screenshots (2x A4 resolution) are also generated, perfect for sharing on LinkedIn, WeChat, or any platform. When Playwright is unavailable, PNG serves as a reliable fallback.

### 4. Resume Diagnosis -- JD Matching Analysis

Upload your target job description (JD) and get an AI-powered diagnosis:

- **Overall match score** (0-100) with grade rating (A/B/C)
- **6-dimension breakdown** -- Hard Skills, Experience, Soft Skills, Education, Keyword Coverage, Resume Quality
- **Keyword gap analysis** -- Which JD keywords are covered vs. missing
- **AI optimization suggestions** -- Prioritized actionable improvements (Critical / Important / Tip)

### 5. JSON Data Persistence

Your resume data is automatically saved as structured JSON:

- **Reusability** -- Load previous resume data to generate new styles or update content without re-entering everything
- **Versioning** -- Each generation includes a timestamp and version number
- **Cross-session** -- Resume data persists across conversations for easy iteration

## How It Works

```
Step 1: Provide Info          Step 2: AI Optimizes        Step 3: You Confirm
|-- Upload / Paste / Chat     |-- Extract & preserve      |-- Review before/after
|-- 5 input methods           |-- Truthful quantification  |-- Adjust or approve
|-- Or use /raw for no edit   |-- Professional rewording   |-- Choose density & style
                              |-- JD keyword extraction & gap analysis (if JD provided)
                              |-- Company attribution lock

Step 4: Pick Style            Step 5: Get Everything
|-- 29 premium designs        |-- Visual resume (HTML + PDF + PNG)
|-- AI recommends 3 best      |-- ATS version + Markdown backup
|-- Preview before final      |-- Cover letter + LinkedIn content
                              |-- Job search strategy report
                              |-- Interview prep report
                              |-- Resume diagnosis report (with JD)
                              |-- JSON data for future reuse
```

**Two Modes:**
- **Default** -- AI optimizes your content (recommended)
- **RAW** -- Zero modifications, design only (`/raw`)

## Quick Start

### For Users

```bash
# In any AI coding assistant
use resume-design-pro
```

Follow the prompts to:
1. Provide your resume information (upload, paste, or conversation)
2. Choose content optimization mode (default) or RAW mode
3. Select from 29 design styles
4. (Optional) Upload target job description (JD) for resume diagnosis and keyword optimization
5. Get your complete resume package

### For Developers

```bash
# Clone and install
git clone <repo-url>
cd resume-design-pro/scripts
npm install

# Export PDF from HTML
node export-pdf.js ../templates/template-01-minimalism.html

# Export PNG fallback (when Playwright unavailable)
node export-png.js ../templates/template-01-minimalism.html
```
