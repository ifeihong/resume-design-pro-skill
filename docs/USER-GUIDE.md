# Resume Design Pro - 用户使用指南

## 快速开始

### 第一步：启动 Skill

在 Trae 的 AI 对话中输入：

```
use resume-design-pro
```

### 第二步：选择信息输入方式

系统会提示 5 种方式：

| 方式 | 适合谁 | 预计时间 |
|------|--------|---------|
| **📄 上传现有简历** | 已有简历想美化/改版 | 5-10 分钟 |
| **💬 对话式问答** | 没有简历或想重新整理 | 15-20 分钟 |
| **📝 直接粘贴** | 有 LinkedIn/草稿想转换 | 3-5 分钟 |
| **⚡ 快速模式** | 紧急需要或测试效果 | 2-3 分钟 |
| **📝 原封不动模式** | 内容已完善，只想换设计 | 3-5 分钟 |

**重要提示：内容处理模式**

| 模式 | 说明 | 适合场景 |
|------|------|----------|
| **默认模式（推荐）** | AI 自动优化内容：筛选重点、量化成果、专业措辞、优化结构 | 大多数用户 |
| **原封不动模式** | AI 不做任何内容修改，仅做排版和格式转换 | 内容已完善，只想换设计风格 |

切换方式：
- 初始菜单选择 `[5] 原封不动模式`
- 任何阶段输入 `/raw` 切换
- 优化确认环节回复 "原封不动"

## 详细使用教程

### 方式 1：上传现有简历

**操作流程：**

1. 选择 `[1] 📄 Upload existing resume`
2. 将你的简历文件（PDF/Word/图片）放到项目目录
3. 告诉 AI 文件路径，例如：
   ```
   我的简历在 D:\Documents\my-resume.pdf
   ```
4. AI 会解析并展示提取的信息表格
5. 确认信息无误，或告诉 AI 需要修改的地方
6. 选择设计风格
7. 生成简历

**示例对话：**
```
AI: 请提供你的简历文件路径

你: D:\Documents\我的简历.pdf

AI: [解析完成] 我已提取到以下信息：
    ┌──────────┬─────────────────┐
    │ 姓名     │ 张三            │
    │ 电话     │ 138****8888     │
    │ ...      │ ...             │
    └──────────┴─────────────────┘
    请确认是否正确，或告诉我需要修改的内容。

你: 工作经历漏了第二段，我在 XX 公司也工作过 2 年

AI: [补充信息] 已添加。现在请选择设计风格...
```

### 方式 2：对话式问答（推荐）

**操作流程：**

1. 选择 `[2] 💬 Guided conversation`
2. 按模块依次回答 AI 的问题
3. 每个模块回答完后，AI 会确认并进入下一模块
4. 可随时说"跳过"跳过可选模块
5. 全部完成后选择风格
6. 生成简历

**模块顺序：**
```
Step 1/9 - 个人信息（姓名、电话、邮箱等）+ 头像选择
Step 2/9 - 职业摘要（2-4 句话概括）
Step 3/9 - 求职意向（目标职位、行业）
Step 4/9 - 工作经历（公司、职位、成果）
Step 5/9 - 教育背景（学校、专业、学历）
Step 6/9 - 技能专长（分类列出）
Step 7/9 - 项目经验（可选）
Step 8/9 - 证书荣誉（可选）
Step 9/9 - 语言能力（可选）
```

**头像选择（内容收集完成后）：**

头像选择在简历内容解析完成后进行，AI 会：

1. **自动检测**：如果你上传了 PDF/Word 简历，AI 会自动检测其中是否有头像
2. **智能推荐**：根据你的目标岗位推荐合适的头像类型

| 选项 | 说明 | 适合场景 |
|------|------|----------|
| 📷 使用原简历头像 | 自动提取你上传简历中的照片 | 已有专业形象照 |
| 📷 上传真实照片 | 提供专业形象照 | 金融/咨询/高管岗位 |
| 🤖 AI 生成头像 | 根据描述生成专业形象照 | 无现成照片 |
| 🎨 抽象图形 | 根据简历风格自动生成几何图形 | 设计/创意岗位 |
| ❌ 不使用头像 | 纯文字排版 | 部分外企/技术岗位 |

> 💡 头像可随时输入 `/avatar` 重新选择

**示例对话：**
```
AI: Step 1/9 - 个人信息
    请告诉我：
    - 你的全名
    - 电话号码
    - 邮箱地址
    - 所在城市（可选）
    - LinkedIn/GitHub/个人网站（可选）

你: 张三，138-0000-8888，zhangsan@email.com，上海

AI: [记录完成] 进入 Step 2/9 - 职业摘要
    请用 2-4 句话描述你的核心竞争力和职业定位...
```

### 方式 3：直接粘贴

**操作流程：**

1. 选择 `[3] 📝 Paste text directly`
2. 从你的 LinkedIn、Notion、或其他地方复制简历文本
3. 粘贴到对话中
4. AI 会自动解析并结构化
5. 确认并补充
6. 选择风格，生成简历

**示例对话：**
```
AI: 请将你的简历文本粘贴到下方

你: [粘贴一大段简历文本]

AI: [解析完成] 我已结构化你的简历内容：
    [展示结构化结果]
    是否有需要补充或修改的？
```

---

### 方式 4：快速模式

**操作流程：**

1. 选择 `[4] ⚡ Quick mode`
2. 只回答 6 个核心问题
3. AI 自动推荐风格并生成

**需要的信息：**
```
1. 姓名 + 联系方式
2. 目标职位
3. 3 句话职业摘要
4. 最近一份工作（公司+职位+1个亮点）
5. 最高学历
6. Top 5 技能
```

---

### 方式 5：原封不动模式

**操作流程：**

1. 选择 `[5] 📝 原封不动模式` 或在任何阶段输入 `/raw`
2. 按上述任意方式提供信息
3. AI 仅做排版和格式转换，不做任何内容修改
4. 选择风格，生成简历

> 💡 如需恢复优化模式，输入 `/optimize`

## 内容优化说明

### 默认模式优化内容

AI 会自动进行以下优化，**所有优化基于你的真实信息，绝不编造数据**：

| 优化项 | 说明 | 示例 |
|--------|------|------|
| 信息筛选 | 保留全部工作经历，按相关性排序 | 5 段经历全部保留，核心经历详细展开 |
| 成果表达 | 基于真实数据的三种处理方式 | 精确数据保留、模糊数据标注、无数据改用定性描述 |
| 措辞优化 | 动词开头 + STAR 法则 | "负责运营" → "独立运营，策划内容并设计互动活动" |
| 结构重组 | 按专业简历标准排列 | 工作经历倒序、技能分类、公司归属锁定 |

### 真实量化策略（三档处理）

| 档位 | 你的数据情况 | AI 处理方式 | 示例 |
|------|-------------|------------|------|
| **A档** | 提供了具体数字 | 保留并优化表达 | "做了3个项目" → "主导 3 个核心项目" |
| **B档** | 提供了范围/大概 | 保留原意，标注估算 | "做了几十个项目" → "完成约 20+ 个项目（估算）" |
| **C档** | 未提及任何数字 | 绝不编造，改用定性描述 | "提升了效率" → "显著优化工作流程，获得团队认可" |

**AI 绝不会：**
- ❌ 将 "做了很多" 改成 "主导 15+ 项目"
- ❌ 将 "提升了效率" 改成 "效率提升 40%"
- ❌ 将 "团队很大" 改成 "管理 12 人团队"

### 内容密度选择

根据你的工作年限，AI 会推荐合适的密度级别：

| 密度 | 适合 | 亮点数量 | 页数 |
|------|------|---------|------|
| **精简版** | 5 年以下经验 | 每段 2-3 个亮点 | 1 页 |
| **标准版** ⭐ | 5-10 年经验 | 每段 3-4 个亮点 | 1-2 页 |
| **详细版** | 10 年以上/高管 | 完整展示所有亮点 | 2-3 页 |

### 优化确认环节

AI 会在生成前展示优化对比，明确标注数据来源：

```
【工作经历 - XX公司】（原始来源：你提供的简历第 X 页）
优化前：[原始描述]
优化后：[优化后描述]

⚠️ 注意：以下数据为你提供的范围估算，如需精确数字请补充
```

你可以选择：
- **确认**（推荐）→ 使用优化后的内容，继续选择设计风格
- **修改** → 告诉 AI 需要调整的地方，例如："第三段经历的数字不对"
- **原封不动** → 使用原始内容，不做任何优化

## 输出文件说明

生成后你会得到以下文件：

| 文件 | 用途 | 怎么用 |
|------|------|--------|
| `resume-姓名-风格.html` | 设计版 | 浏览器打开，Print → Save as PDF |
| `resume-姓名-ats.html` | ATS 版 | 投递招聘系统时使用 |
| `resume-姓名.md` | 备份 | 纯文本，随时编辑复用 |
| `cover-letter-姓名-公司.html` | 求职信 | 针对特定公司定制（可选） |
| `linkedin-export.md` | LinkedIn 优化 | 复制粘贴到 LinkedIn 各模块（可选） |

### 如何导出 PDF

**方法 1：浏览器打印（推荐 - 最简单）**

1. 用浏览器打开 `resume-姓名-风格.html`
2. 按 `Ctrl + P`（或 `Cmd + P`）
3. 目标打印机选择 "另存为 PDF"
4. 纸张大小选择 A4
5. 边距选择 "默认"
6. 勾选 "背景图形"
7. 保存

**方法 2：Node.js 脚本（批量导出）**

```bash
# 安装依赖（首次）
npm install
npx playwright install chromium

# 导出单个简历
node scripts/export-pdf.js resume-zhangsan-cyberpunk.html

# 批量导出所有简历
node scripts/export-pdf-batch.js *.html

# 导出到指定文件夹
node scripts/export-pdf-batch.js *.html --output-dir ./pdf-output
```

## 如何修改/更新简历

### 方法 A：重新生成（推荐大改动）

```
use resume-design-pro
→ 选择任意输入方式
→ 提供更新后的信息
→ 选择相同或不同风格
```

### 方法 B：直接编辑 HTML（小改动）

1. 找到生成的 `.html` 文件
2. 用任意文本编辑器打开（VS Code、记事本等）
3. 找到要修改的文字，直接替换
4. **注意**：不要修改 CSS 类名，只改文本内容
5. 保存后在浏览器刷新查看效果

**示例：修改工作经历**
```html
<!-- 找到这段 -->
<div class="exp-company">字节跳动 · 抖音电商团队</div>

<!-- 改成 -->
<div class="exp-company">腾讯 · 微信事业群</div>
```

### 方法 C：告诉 AI 具体修改

```
"在我的简历里添加一段新工作经历：
  公司：XX科技有限公司
  职位：技术负责人
  时间：2024.01 - 至今
  亮点：带领 10 人团队完成核心系统重构"

"把我的技能从 React 改成 Vue"

"换成 Minimalism 风格"
```

## 常见问题

**Q: 我没有头像照片怎么办？**
A: 可以选择不上传头像、AI 生成头像、或使用抽象几何头像。随时输入 `/avatar` 重新选择。

**Q: 生成的简历是中文还是英文？**
A: 根据你输入的语言自动适配。你也可以指定：`/lang en` 或 `/lang zh-CN`。

**Q: 可以做多页简历吗？**
A: 默认单页（A4）。如果需要多页，告诉 AI "我需要两页版本"。

**Q: 字体需要安装吗？**
A: 不需要。所有字体从 Google Fonts 在线加载，有网络即可。

**Q: 可以在手机上查看吗？**
A: 可以。所有模板都是响应式设计，手机、平板、电脑都能正常显示。

**Q: 投递时 HR 能看到设计效果吗？**
A: 建议同时提供 ATS 版本。设计版用于邮件附件或在线展示，ATS 版用于招聘系统投递。

**Q: 什么是 ATS 版本？**
A: ATS（Applicant Tracking System）是招聘系统。ATS 版本使用语义化 HTML、纯文本内容、标准 section 层级，确保招聘系统能正确解析你的信息。

**Q: LinkedIn 导出怎么用？**
A: 生成后打开 `linkedin-export.md`，按照里面的模块（Headline、About、Experience 等）逐一复制粘贴到你的 LinkedIn 个人资料中。每个模块都有优化建议。

**Q: 求职信怎么生成？**
A: 在简历生成完成后，告诉 AI "生成求职信"，并提供目标公司名称。AI 会生成与简历风格匹配的求职信。
